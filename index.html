TitleBug: [BXH][Network][Data] Leaderboard data not updating

Environment: Web/Apps | Version 1.0.0

Steps to Reproduce (EN):

  • Navigate to "BXH" (Leaderboard) section.
  • Observe the displayed ranking data.
  • Wait for an update cycle or refresh the page.

Steps to Reproduce (VN):

  • Truy cập vào mục "BXH".
  • Quan sát dữ liệu xếp hạng hiển thị.
  • Chờ đến chu kỳ cập nhật hoặc tải lại trang.

Actual Result: "BXH" data does not update. Notification "Không thể kết nối máy chủ. Xin thử lại." appears.

Evidence: Picture or video link will be placed here.

Expected Result: "BXH" data should update in real-time or upon refresh. No error should appear unless there's a real server issue.

Priority: High (Affects real-time ranking updates and user experience)
